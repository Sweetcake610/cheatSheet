Font Styling

Property Description
font-family Changes the typeface, font, of the text on the webpage.
@font-face Imports outside fonts directly into your stylesheet for the webpage. In order to use this property you must link the outside font url to your html document, extract rules labeled /*latin*/ from the url link and then paste the extracted rules to the style.css file.
font-size Changes the size of the text on the webpage. Property requires a unit of measurement such as pixels (px), em, or percentages (%).
font-weight Changes the weight of the font on the webpage. Accepts values of normal, bold or even number values from 100 - 900.
font-style Changes the style of the text on the webpage. Accepts values of normal, and italic.
text-transform Text can be styled to appear in either all upercase or lowercase.
text-align To move or align text using the following values: left, right, center, or justify.
text-decoration Sets the appearance of decorative lines on text. Some of the values that are used are underline, overline, line-through, or none.
line-height To avoid creating text that is difficult to read, you can modify the spacing between lines of text with the link-height property. Preferred em measurements, but can also use pixels (px).
word-spacing Increases the spacing between words in a boyd of text. Preferred em measurements.
letter-spacing Increasing the spacing between individual letters, the technical term for the spacing between letters is the kerning. Preferred em measurements.

Colors

Property
color

The foreground color

 background-color

The background color
Description
Colors can be represented in one of these options listed below
named colors CSS has a total of 147 named colors.
RGB colors Represented in CSS by three numbers separated by commas in RGB order. Numbers range from 0 - 255.
Hexadecimal Colors Hex values are just a different way to represent RGB color. Instead of three numbers you are using six hexadecimal numbers [0-9][A-F]
HSL colors HSL stands for Hue, Saturation and Lightness. Hue (color wheel representation) , Saturation (amount of gray in a given color), and Lightness (amount of white in a given color)
opacity Opacity is a measure of how transparent a color is. This can be added to the end of your RGB or HSL, represented with an "a"

Box Model

Property Description
Width & Height An elements box has two dimensions: a width and a hight. All boxes have default dimensions that are automatically set to hold the raw contents of the box.
Margin The margin refers to the space directly outside of the box.
Padding The space between the contents of a box and the borders of a box is known as padding.
Border The border property defines the width, line style and color of the border of an elements box.
Visibility Elements can be hidden from view with the visibility property. It accepts two value: hidden or visible
Overflow The overflow property controls what happens to content when it spills, or overflows outside of its box.
