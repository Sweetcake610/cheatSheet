CSS: Cascading Style Sheets

What is CSS?

Css is a language that web developers use to style the html content on a web page.

Style Tag (Internal Stylesheet)

Although CSS is a different language from html, we can write CSS code directly into html file by using the style tag.

Exp: <style>.......</style>

The style element allows you to write CSS code between its opening and closing tags. The style element must be placed inside the head of the html file.

The .CSS File

Putting both HTML and CSS coding in one file can cause confusion for web developers and makes coding harder to read easily. So you can create a CSS file by using the .CSS file name extension, like so:

Exp: style.css

When placing CSS code in a CSS file, in order for it to apply to the HTML file you have to link the HTML file and the CSS file together.

Font Styling

Property Description
font-family Changes the typeface, font, of the text on the webpage.
@font-face Imports outside fonts directly into your stylesheet for the webpage. In order to use this property you must link the outside font url to your html document, extract rules labeled /*latin*/ from the url link and then paste the extracted rules to the style.css file.
font-size Changes the size of the text on the webpage. Property requires a unit of measurement such as pixels (px), em, or percentages (%).
font-weight Changes the weight of the font on the webpage. Accepts values of normal, bold or even number values from 100 - 900.
font-style Changes the style of the text on the webpage. Accepts values of normal, and italic.
text-transform Text can be styled to appear in either all upercase or lowercase.
text-align To move or align text using the following values: left, right, center, or justify.
text-decoration Sets the appearance of decorative lines on text. Some of the values that are used are underline, overline, line-through, or none.
line-height To avoid creating text that is difficult to read, you can modify the spacing between lines of text with the link-height property. Preferred em measurements, but can also use pixels (px).
word-spacing Increases the spacing between words in a boyd of text. Preferred em measurements.
letter-spacing Increasing the spacing between individual letters, the technical term for the spacing between letters is the kerning. Preferred em measurements.

Colors

In CSS its important to make two distinctions about color. Color can affect the following design aspects.

The foreground color

Exp: color: red;

The background color

Exp: background-color: red;

Foreground color is the color that an element appears in. The background color is the color that an elements background appears in.
CSS has a total of 147b named colors. YOu can google this information without having to remember. To take advantage of the full spectrum of colors that CSS supports you have the option of using RGB colors. There are 16,777,216 possible colors.

RGB Colors

RGB colors are represented in CSS by three numbers separated by commas in that order. RGB colors can be used for both foreground and background colors.

Exp: p { color: rgb(123, 20, 233); }

Online resources are available for correct number combo.

Hexadecimal Color Codes

Often referred to as hex color codes for short, Hex color codes also offer 16,777,2216 options, but they follow a different syntax. with RGB color mixture the values are in base 10, hex color codes use base 16 or hexdecimal base.

Exp: RGB (0--256, 0--256, 0--256) and HEX (#[0-9][A-F])

Exp: h1 { color: #09AA34; }

When read from left to right, each group of two characters responds to a value for red, green and blue. All hex color codes begin with a '#' character.
When a hex code is composed entirely of the same characters, the hex color can be abbreviated.

Exp: h1 {color: #FFFFFF; or color: #FFF; }

Box Model

Property Description
Width & Height An elements box has two dimensions: a width and a hight. All boxes have default dimensions that are automatically set to hold the raw contents of the box.
Margin The margin refers to the space directly outside of the box.
Padding The space between the contents of a box and the borders of a box is known as padding.
Border The border property defines the width, line style and color of the border of an elements box.
Visibility Elements can be hidden from view with the visibility property. It accepts two value: hidden or visible
Overflow The overflow property controls what happens to content when it spills, or overflows outside of its box.
