Intro to HTML

What is HTML?

HTML = HyperText Markup Language
A markup language is a computer language that defines the structure and presentation of raw text.
HyperText is text displayed on a computer or device that provides access to other text through, links, also know "hyperlinks".

!DOCTYPE html

A web browser must know what language a document is written in before they can process the contects of the document.
To do this you need a:

Document Type Declaration
HTML use <!DOCTYPE html>

(This must be the frist line of code in all html documents)
This declaration is an instruction. It tells the browser what type of document to expect and what version of html is being used.
The DOCTYPE is used for html5. Other versions like html4 or Xhtml 1.0 have different DOCTYPES.

Preparing HTML Dcouments

To begin adding HTML structure and content, we must first add opening and closing tags. Without these tags it's possible that browsers could incorrectly interpret your html code.

<html>Enter code here </html>

HTML Anatomy

  1. Angle Brackets [< >]
  2. Html element or element [code inside the angle brackets]
  3. Opening Tag [html tag used to start an html element]
  4. Closing Tag [html tag to end an html element. Closing tags have a forward slash inside them.]

<p> Hello World! </p>

Head Element

The head element will contain information about the page that isn't displayed directly on the actual web page, which is also called the metadata.
