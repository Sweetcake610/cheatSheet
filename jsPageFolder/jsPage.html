Introduction to Javascript

JavaScript is a widely used web-based programming language that powers the dynamic behavior on most websites.

JavaScript allows you to make web pages more interactive by accessing and modifying the content and markup used in a web page while it is being viewed in the browser.

What is Script?

A script is a series of instructions that a computer can follow to achieve a goal.

To write a script, you need to first state your goal and then list the tasks that need to be completed in order to achieve it. So a program must give the computer enough detail to perform the task as if every time were it first time.

Start with the big picture of what you want to achieve and break that down into smaller steps.

  1. Define the Goal
  2. Design the Script
  3. Code each Step
Often scripts will need to perform different tasks in different situations. You can use flowcharts to work out how the tasks fit together.

Variable Reference

A variable is data that is tored in it temporarily. Variables are declared by programmers. Programmers can declare variable by using var, let or const.
var data = 'I am data';
let data = 'I am data';
const data = 'I am data';

Primitive Data Types Reference

Data types are the building blocks of all languagues and essential pieces of any program.
Strings Any grouping of keyboard characters surrounded by single or double quotes.
Numbers Any number, including numbers with decimals.
Boolean Either true or false with no quotations.
Null Equivalent to zero. Can only be null.
Undefined Represents the absense of a value, the given value does not exist.

String Methods and Properties

List of String Methods and Properties
Property Description
data.length(); Returns the number amount of characters in a string
data.indexOf(); Returns the index location of the specified text.
data.lastIndexOf(); Returns the index of the last occurrence.
data.search(); Returns the position of the specified value
data.slice(start, end); Returns a string with the set start index position and end index position.
data.replace(); Replaces your previous string with a new string.
data.toLowerCase(); Returns a string with all lowercase characters.
data.toUpperCase(); Returns a string with all uppercase characters.

Number Methods and Properties

List of Number Methods and Properties
Property Description
(10).toString(); Returns the number as a string
(10).toExponential(); Returns a number written in exponential notation.
(10).toFixed(); Returns a number written with a number of decimals.
(10).valueOf(); Returns a number as a number.
